Routing

Data routes are variable-routes that use URL segment expressions to define a mechanism for passing data via URL in a formatted fashion.

Data Routes

This router supports segment and query-string variables and we provide a simple, declarative method for using that data in your application. By using special (Node.js Express-based) notation in the <x-view> element's URL attribute, the route data becomes available in our unified data expression system.
