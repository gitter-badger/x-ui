Routing

Content routing is a simple declarative approach for creating multi-page applications from a single HTML file. The browsers URL path will determine what content is be displayed.

Nested Routes

Nested routes - ir routes and sub-routes are defined statically. You can choose to semantically nest <x-view> elements and have their path's automatically inherited from the parent or declare them in a flat, explicit format.
