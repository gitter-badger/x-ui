## Install with NPM

### 1. Install the NPM package.

````bash
  npm i @viewdo/x-ui
````

### 2. Add the script tags to the head

````html
<head>
  ...
  <script type="module" src="/node_modules/@viewdo/ui/dist/viewdo.esm.js"></script>
  <script nomodule src="/node_modules/@viewdo/ui/dist/viewdo.js"></script>
  ...
</head>
````
