view.DO's Experience UI is the first HTML-only tool-set built specifically for interactive web experiences.

Create engaging content without compilation or a framework; and in most cases, without any JavaScript at all.

These documents are created using these components. View the source to see how it has all been done without script. The HTML alone, declaratively creates a single-page app, with complex routing, guided navigation with decision-trees, automatic presentations and content data-binding using any data source.

How does this help?

This set of inter-working elements make it simple to build engaging, application-like experiences with dynamic-content, guided navigation, timed-actions, voice-over audio, video, hooks for animations and much more.

These components use a convention-over-configuration approach to guide user behavior with your content in a rule-based, presentation format.

Smart Guided Navigation

  • Automatic Back & Forward Resolution
  • Sequenced Content: present, educate, inform
  • Sequenced Input: surveys, quiz and display results
  • Sequenced Video: interactive video story-telling
  • Declarative: Expression-based Conditions
  • Stateful: pick up where a user left-off

Built for Presentations

  • Timed Ambient & Voice-Over Audio

Dynamic & Personalized

  • Personalize the content for each of your visitors.
