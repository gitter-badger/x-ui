What is <x-ui/>?

<x-ui/> (aka, experience UI) is the first UI component toolkit built specifically for turning static content into interactive web experiences.

This documentation was created with <x-ui/>. See how complex routing, guided navigation, automatic presentations, data-binding and more work when you take the tour. Click to begin.

Pure HTML

<x-ui/> components make single-page apps simple! We created special elements for expressing routes, navigation, dynamic-content, data-binding and much more, using HTML semantics and no JavaScript.

PLUS we use a convention-over-configuration approach to dramatically simplify everything. There is no longer a need to be an experienced web developer to create robust web experiences - if you know HTML and CSS you can start building NOW.

How does this help?

For most developers, building a web-app begins by choosing an app framework or static site generation tool. These tools are often complex and time-consuming to learn, set up and operate. Once a framework is adopted, you are locked-in and it's difficult to change direction, without a costly endeavor.

Frameworks exist to minimize the effort to create and maintain web pages. They do that well, but at a high costs and a long-term commitment not suited for every use-case.

No Framework, No Problem

<x-ui/> is different. Without transpilation, a framework or even a build-process, developers get the same time-saving techniques (layouts, routing, data-binding, etc) using specialized web components directly in HTML. They are so easy to bring into any project; these tools can be quickly added to any site or page with a simple script reference. No need to check if your site is compatible -- if you can inject a <script> tag, you can start using these today.

Features

  • Content Routes w/Lazy Loaded HTML
  • Support for Data Routes for Data-Driven Content
  • Use Route Data Directly in HTML w/Route and URL Data Providers

  • Automatic Back & Forward Resolution
  • Sequenced Content: present, educate, inform
  • Sequenced Input: surveys, quiz and display results
  • Sequenced Video: interactive video story-telling
  • Declarative: Expression-based Conditions
  • Stateful: pick up where a user left-off

  • Data Tags to render personalized data from a variety of sources
  • Use HTML to access data from your API
  • Capture attention and data, with smart workflows to replace boring forms.
  • Personalize the content for each of your visitors.

  • Interactive Videos
  • Queued, Timed & Event-based Actions
  • Animated Transitions
  • Background Music & Voice-Overs
  • Animation Hooks

  • Fast: Lightweight and lightening fast: No Framework & Lazy Loaded
  • Themeable: BYO UI and themes. <x-ui/> stays out of the way.
  • Extensible: BYO Data with a plugin system to easily connect your data
  • Typed: Built with TypeScript so you can enjoy completely typed components
  • Portable: Native web components with hooks to integrate with any framework
  • Trackable: Analytics & Behavior Tracking Plugins
  • Extensible: The provider system makes it easy to integrate with anything.
